👥 Control de Acceso por Rol

Tres roles con permisos diferenciados: admin, staff y bioquimico.

👑 Admin

Acceso total al sistema. Único que puede gestionar usuarios y prestaciones.

  • ✅ Dashboard + Calendarios
  • ✅ Pacientes + Ingresos
  • ✅ Resumen médico
  • ✅ Listados (todos)
  • ✅ Facturación OS
  • ✅ /usuarios → Crear/editar/eliminar usuarios
  • ✅ /prestaciones
  • ✅ Auditoría completa
  • ✅ Reportes estadísticos

🧑‍💼 Staff

Personal administrativo. Acceso completo excepto gestión de usuarios y prestaciones.

  • ✅ Dashboard + Calendarios
  • ✅ Pacientes + Ingresos
  • ✅ Resumen médico
  • ✅ Listados (todos)
  • ✅ Facturación OS
  • ✅ Perfil propio
  • ❌ /usuarios
  • ❌ /prestaciones

🧪 Bioquímico

Personal de laboratorio. Acceso limitado a 3 rutas. Redirigido a /ingresos si accede a otras.

  • ✅ /ingresos (lectura)
  • ✅ /pacientes (lectura)
  • ✅ /perfil propio
  • ❌ Crear ingresos (bloqueado en backend)
  • ❌ Calendarios
  • ❌ Facturación
  • ❌ Listados
  • ❌ Resumen médico
